# Project Screenshots

Add your project screenshots here with the following names:

- `investimate.png` - Screenshot of Investimate.ai (https://investimate-ai-eight.vercel.app/)
- `quickvid.png` - Screenshot of QuickVid (https://quick-vid-two.vercel.app/)
- `bookery.png` - Screenshot of Bookery (https://bookster-53qm.onrender.com)

## How to get screenshots:

1. Visit each live site
2. Take a full-page screenshot (or the main hero/dashboard section)
3. Save as PNG format with the names above
4. Place them in this folder

## Recommended dimensions:
- Width: 1200px - 1920px
- Height: 600px - 1080px
- Format: PNG or JPG
- Optimize images for web (keep file size under 500KB)

The portfolio will automatically display these images in the project cards!
